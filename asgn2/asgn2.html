Instructions

  • Mouse Drag: Click and drag on the canvas to rotate the view
  • Shift + Click: Make the rat jump and spin!
  • Global Rotation: Use the slider to rotate the entire rat
  • Joint Sliders: Control the front left leg joints (works when animation is OFF)
  • Animation: Toggle walking animation with the buttons

Grading Rubric Checklist

  • ✅ Draw a Cube: Functional drawCube(Matrix M) using triangles.
  • ✅ Global Rotation: Slider and uniform u_GlobalRotateMatrix implemented.
  • ✅ renderScene(): Centralized drawing function handles all rendering.
  • ✅ 8+ Parts: Rat has ~16 parts (Body, Head, Snout, Ears, Eyes, Legs, Tail).
  • ✅ Joint Sliders: Control for Front Left Leg (Upper, Lower, Foot).
  • ✅ Hierarchy: At least one chain extends 3 elements deep (Legs and Tail).
  • ✅ tick(): Animation loop using requestAnimationFrame.
  • ✅ Animation Toggle: Buttons to enable/disable movement.
  • ✅ Color: Custom colors for all body parts using uniforms.
  • ✅ Natural Animation: Coordinated leg swing, tail wag, and body bob.
  • ✅ Non-Cube Primitive: Cylinder used for the tail segments.
  • ✅ Shift-Click Poke: "Startled Rat" jump/shake sequence.
  • ✅ Mouse Control: Interactive drag-to-rotate on X and Y axes.
  • ✅ Performance: Optimized with static buffers (>= 10 FPS).
  • ✅ FPS Indicator: Real-time performance display on canvas.
